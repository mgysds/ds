今日大赛-每日大赛-今日大赛-今日活

　走在路上，旁边的一户人家原本关着的大门陡然打开，从里面涌出一排丫鬟。
　　一声声哭泣声从院子里传出，很多人都是满脸泪痕。
　　不多时，一阵唢呐声传来，道路的另一边，一队人马运送着棺材赶来，停在了这户人家的门口。
　　大家手忙脚乱的将一个妇人从屋内运出，准备送入棺材。
　　“看样子陈员外的老婆没能挺过去。”
　　“一尸两命，可怜呐。”
　　“陈员外夫妻两个人都很好，可惜了。”
　　周围的人纷纷摇头叹息，但对这种现象早已见怪不怪。
　　修仙世界的医疗技术跟李念凡前世的古代相差无几，尤其是生孩子时，除非有仙人相助，否则极容易出事。
　　李念凡的内心也是一阵波动，他的目光落在妇人的身上，见她双眸紧闭，似乎已经没了呼吸，但额头上依然有汗水溢出。
　　只一眼，他就看出，这妇人并没有死亡，而是虚脱到昏厥。
　　“等等！她还没死！”
　　李念凡看着即将合上的棺材，连忙上前阻止。
　　一位赤红着眼睛的中年人声音沙哑道：“李公子，你做什么？”
　　喜事变丧事，如果不是李念凡在落仙城口碑很好，他早就翻脸了。
　　“你老婆和儿子都没死，我可以救他们！”李念凡说道。
　　“李公子，你没有开玩笑？”中年人愣住了。
　　周围的产婆也同样不可思议的看向李念凡。
　　“我确定能救！”李念凡急切道：“你赶紧疏散人群，再让人准备纱布和油灯，我来救人！时间来不及了！”
　　情况危急，他只能在棺材里救人了。


第十五章 或许这就是返璞归真吧
　　李念凡现在非常庆幸，系统虽然走了，但系统空间还在，里面还放着手术用的各种道具。
　　距离此处不远处的阁楼之上，两男两女正在一起用餐。
　　他们不是别人，正是洛诗雨、国师的儿子以及丞相的儿子，还有另一名女子则是洛诗雨的贵客，凌云仙阁的圣女林清云。
　　丞相和国师的儿子正在对洛诗雨大献殷勤，时不时还相互挑衅，妥妥的舔狗争宠的画面。
　　圣皇钦点洛诗雨嫁给国师的儿子，然而，洛诗雨却又和丞相的儿子走得很近，奇特的三角关系，让丞相和国师之间的关系变得非常紧张起来。
　　虽然国师的儿子百般殷勤，但洛诗雨的态度明显偏袒于丞相的儿子，让丞相一方士气大振。
　　据说，丞相已经上表奏章，说公主和自己的儿子才是情投意合，希望圣皇取消公主和国师儿子的赐婚，转而把公主嫁给自己的儿子，成就一段佳话。
　　这无疑让丞相和国师之间的矛盾飞速升级，双方矛盾不断激化。
　　洛诗雨眼见李念凡的计划非常顺利的实施，心头大振，对李念凡更为的感激。
